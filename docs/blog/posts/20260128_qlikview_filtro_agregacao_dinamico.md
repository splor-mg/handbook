# 🚀 Qlikview - Menu de agregação de valores dinâmico

Neste treinamento, é demonstrado como implementar menus de agregação de valores dinâmicos e interativos no QlikView utilizando variáveis e expressões condicionais. A abordagem permite controlar dinamicamente a exibição de colunas, cores e comportamentos de objetos em relatórios.

<!-- more -->

O vídeo apresenta o conceito de criação de um menu “casca”, ou seja, um objeto que não está associado diretamente ao modelo de dados, mas que serve como entrada para variáveis usadas em fórmulas. A partir dessas variáveis, o QlikView interpreta valores como 1 (verdadeiro) e 0 (falso) para decidir, por exemplo, se uma coluna deve aparecer ou desaparecer em uma tabela.

Também é mostrado como:

Usar ações de botão para alterar valores de variáveis;

Aplicar condições de exibição de colunas com base nessas variáveis;

Alterar cores de objetos dinamicamente, utilizando expressões condicionais e valores RGB;

Criar menus visuais que indicam o item selecionado por meio de destaque de cores.

O treinamento reforça que a lógica é simples, baseada em variáveis e condições, mas extremamente poderosa, permitindo alto nível de customização de dashboards no QlikView. Ao final, é destacada a possibilidade de replicar conceitos semelhantes em outras ferramentas, como o Power BI, ainda que com implementações diferentes.
